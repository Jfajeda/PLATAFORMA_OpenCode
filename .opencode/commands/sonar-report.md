Consulta el estado actual del proyecto en SonarCloud y genera un reporte.

1. Usa la herramienta sonarqube del MCP server para obtener las metricas del proyecto.
2. Muestra el estado del Quality Gate (PASSED/FAILED).
3. Lista los issues abiertos agrupados por tipo (Bug, Vulnerability, Code Smell).
4. Para cada issue critico o bloqueante, muestra el archivo y la linea afectada.
5. Sugiere un plan de accion para resolver los issues por orden de prioridad.
